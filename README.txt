Application Web permettant de consulter l’état des stations Vélib' en temps réel et de louer fictivement un vélo en remplissant un formulaire. 

Réalisations :

-	Développement d’un Template intégrant la carte de Google Maps 
-	Utilisation des jeux de données d’Open Data Paris sur la carte.
-	Enregistrement des réservations dans le navigateur et dans un journal
