Présentation du projet :

- Application simulant la réservation de vélos (Velib') dans Paris
- Développement en JavaScript/Jquery
- Utilisation de l'API GoogleMaps en intéraction avec l'API OpenData de Paris
- Utilisation d'autres API : Google Fonts...
- Utilisation de Bootstrap pour la mise en page et l'intégration d'un plugin Jquery (le carousel)
