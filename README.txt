Application Web permettant de consulter l’état des stations Vélib' en temps réel et de louer fictivement un vélo en remplissant un formulaire. - Développement de l’interface utilisateur en Html/Css/ Bootstrap et JQuery
- Utilisation de l'API Google Maps en interaction avec l'API Open Data de Paris
- Utilisation de l’API Web Storage pour les sessions.
- Création d’un fichier Log en PHP
